# Digital Dice Simulator (Pygame)

PythonとPygameライブラリを用いて開発した、キーボード操作によるデジタルサイコロシミュレーターです。

## 概要
物理的なサイコロを振る代わりに、PC上でランダムな出目をシミュレートします。画像を動的に切り替えることで、直感的なビジュアル・フィードバックを提供します。

## 主な機能と技術的特徴
*   **動的な画像ロードと管理**:
    `f-string` とリスト内包表記を活用し、1から6までのサイコロ画像を効率的に一括ロードしています。
*   **インデックス制御による出目管理**:
    `random.randint(0, 5)` で生成した整数をリストのインデックスとして利用し、描画する画像を瞬時に切り替えるロジックを実装しています。
*   **イベント駆動型設計**:
    `pygame.KEYDOWN` イベントをフックし、「Rキーでロール」「スペースキーで終了」という直感的なユーザーインターフェースを提供します。
*   **パフォーマンス最適化**:
    `convert_alpha()` を使用したピクセル形式の最適化により、描画処理の負荷を軽減しています。

## 動作環境
*   **言語**: Python 3.x
*   **ライブラリ**: Pygame

## 導入・実行手順
1.  **ライブラリのインストール**
    ```bash
    pip install pygame
    ```
2.  **画像ファイルの用意**
    プロジェクトと同じディレクトリに以下の画像ファイルを配置してください。
    *   `さいころ1.png` ～ `さいころ6.png`
3.  **プログラムの実行**
    ```bash
    python main.py
    ```

## 操作方法
*   **Rキー**: サイコロを振る（ランダムに画像が切り替わります）。
*   **スペースキー**: アプリケーションを終了する。
*   **閉じるボタン**: アプリケーションを終了する。

## システム構成
*   **描画ループ**: 30 FPSに制御されたメインループによる安定した描画。
*   **座標管理**: 画面中央（300, 300）へのアンカー配置。

## 今後の拡張予定
*   **ロールアニメーション**: Rキーを押した際に、一定時間高速で画像が切り替わる演出の追加。
*   **サウンドエフェクト**: サイコロが止まった時の効果音の実装。
*   **履歴表示**: 過去に出た目のログを表示する機能。
